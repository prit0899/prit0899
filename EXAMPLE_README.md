## How This Works

This portfolio is automatically updated whenever I:
1. Push changes to any of my project repos
2. Update the `portfolio.yml` file in a project
3. The action runs (daily or on-demand)

The action reads metadata from each project's `portfolio.yml` file and generates these cards automatically.
